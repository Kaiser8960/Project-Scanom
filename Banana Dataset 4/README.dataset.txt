# Banana Leaf Disease > Banana Augmentasi
https://universe.roboflow.com/banana-al4eq/banana-leaf-disease-qesr2

Provided by a Roboflow user
License: Public Domain

This banana leaf spot disease model was trained using yolov8. It is able to identify three types of banana leaf diseases, cordana, sigatoka, pestalotiopsis. It is also able to identify healthy banana leaves. Use model v2 for best performance.
